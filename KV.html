Что такое «система контроля версий»?

Это программное обеспечение, предназначенное для отслеживания изменений, внесённых в файлы или наборы файлов, а также для координации работы нескольких людей над одним и тем же проектом.

К какому типу VSC относится? В чем главное отличие от других типов?

VS Code (Visual Studio Code) — это интегрированная среда разработки (IDE), а не просто текстовый редактор. Главное отличие от других типов редакторов (например, Notepad++, Sublime Text) — это наличие расширений и встроенных функций для разработки, таких как отладчик, поддержка Git, интегрированный терминал, IntelliSense (автодополнение кода) и многое другое.

В чем заключается преимущество использование Git?

Главное преимущество Git — это управление версиями и совместная работа. Он позволяет отслеживать изменения в коде, легко возвращаться к предыдущим версиям, работать в команде над одним проектом без конфликтов, и эффективно управлять разными ветками разработки.

Какие команды Git вы знаете?

git init, git add .(добавление файлов), git commit -m "first commit"(сохранение изменений), git config --global user.email "you@example.com"(указать электронную почту автора), git config --global user.name "Your Name"(указать имя автора), git config --list(проверка добавления автора), git branch -M main(выполняется в случае, если при установке пакета Git на компьютер не была установлена галочка на ветку «main»), git remote add origin «ссылка на репозиторий»(добавляется удаленный путь к репозиторию Git Hub), git commit -m «комментарий», git push
